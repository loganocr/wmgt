# WMGT Onboarding Guide

Welcome to the WMGT (Walkabout Mini Golf Tournament) team! This guide will help you get started based on your role.

## What is WMGT?

WMGT — "The Weekly" — is a weekly tournament for the Walkabout Mini Golf VR game. Players register throughout the week, then compete on Saturday in assigned rooms and time slots. Scores are entered into the system and verified against scorecard photos posted to Discord.

## Weekly Schedule

| Day | Time (Central) | Event |
|-----|----------------|-------|
| Sunday | 4-6 PM | Typically, registration opens for next week's tournament |
| Sunday → Saturday | All week | Players can register and practice the announced courses |
| Saturday | 3:00 PM | Registration closes (1 hour before tee time - 22:00 UTC) |
| Saturday | 4:00 PM (22:00 UTC) | Tournament begins — first time slot tees off |
| Saturday → Sunday | ~24 hours | Tournament runs across multiple time slots |
| Sunday | Afternoon | Tournament closes, results finalized, new week opens |

## Roles

WMGT has three levels of access in the application:

### 👤 Regular Users
- Log in via Discord
- Be familiar with the [tournament rules](tournament_rules.md)
- Register for tournaments
- Enter their own scores
- View results, leaderboards, and history

### 🔍 Contributors

Contributors help verify that player-entered scores match the scorecard photos posted to Discord.

- Log in via Google, or app credentials
- Verify Results page — compare entered scores to scorecard images
- Mark issues (No Show, No Scores, Rule Violations)

**📖 Read:** [Contributor Guide](guide-contributor.md)

### 🏆 Tournament Admins
Tournament Admins run the weekly operations — setting up tournaments, managing registrations, assigning rooms, and closing out each week.

- Log in via Google, or app credentials
- Everything Contributors can do, plus:
- Tournament Admin dashboard
- Session setup and management
- Room assignments
- Player registration management
- Tournament open/close operations

**📖 Read:** [Tournament Admin Guide](guide-tournament-admin.md)

### ⚙️ Administrators

Administrators have full system access for managing users, permissions, and system configuration.

- Log in via Google, or app credentials
- Everything Tournament Admins can do, plus:
- User access management (granting roles)
- Build options and configuration
- System parameters

**📖 Read:** [Administrator Guide](guide-administrator.md)

## Getting Access

To get elevated access (Contributor, Tournament Admin, or Administrator), contact an existing Administrator who can grant your role through the **Administration > Manage User Access** page.

## Key Concepts

Before diving into your role-specific guide, familiarize yourself with these terms:

| Term             | Definition |
|------------------|------------|
| **Season**       | A series of 12 weekly tournaments |
| **Week**         | A single tournament event (e.g., "S19W05" = Season 19, Week 5) |
| **Session**      | The tournament session for a specific week |
| **Time Slot**    | A specific start time during the tournament (e.g., 22:00 UTC, 00:00 UTC) |
| **Room**         | A group of 2-4 players assigned to play together (e.g., "WMGT15") |
| **Easy/Hard Course** | Each week features two courses — one easy, one hard |
| **Scorecard**    | The in-game results showing strokes per hole, **must show timestamp to be valid** |
| **Verification** | Confirming entered scores match the posted scorecard photos |

## The Discord Connection

WMGT relies heavily on Discord for communication:

- **#wmgt-scorecards** — Players post their scorecard photos here after playing
- **#wmgt-chat** — General tournament discussion
- **#staff** — Private channel for admins to discuss issues

The verification bot automatically reads scorecard images from Discord and attempts to match them to player entries. You'll see the results of this automated verification (green/red/grey squares) on the Verify Results page.
