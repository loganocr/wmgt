## Weekly Checklist

### Sunday (Setup)
- [ ] Close previous tournament (if not done)
- [ ] Create new session for next week
- [ ] Set courses (Easy & Hard)
- [ ] Set registration open/close dates
- [ ] Announce courses in Discord

### Saturday (Tournament Day)
- [ ] Registration closes (automatic at 3 PM CT)
- [ ] Assign rooms (~3:30 PM CT)
- [ ] Open rooms (~3:45-3:55 PM CT)
- [ ] Monitor during tournament
- [ ] Handle issues in #staff

### Sunday (Wrap-up)
- [ ] Verify all scores
- [ ] Confirm all rooms checked off
- [ ] Close tournament
- [ ] Set up next week (repeat cycle)

## Verification Quick Reference

### Bot Status Colors
| Color | Meaning | Action |
|-------|---------|--------|
| 🟩 Green | Verified by bot | None needed |
| 🟥 Red | Mismatch found | Manual check |
| ⬜ Grey | Bot couldn't match | Manual check |
| (none) | No attempt | Click Verify button, then manual check |

### Issue Actions
| Button | When to Use |
|--------|-------------|
| **No Show** | Player didn't show up |
| **No Scores** | 4+ hours passed, no entry |
| **Rule Violation** | DNF, rage quit, etc. |

### Verification Steps
1. Compare Sum columns to Score columns
2. Click player name to check hole-by-hole
3. Fix transposed numbers if needed
4. Click Save Round after changes
5. Mark Verified checkbox
6. Confirm room checkmark appears

## Glossary

| Term | Definition |
|------|------------|
| **Season** | 12-week tournament series (e.g., S19 = Season 19) |
| **Week** | Single tournament (e.g., S19W05 = Season 19, Week 5) |
| **Session** | Database record for a week |
| **Time Slot** | Scheduled tee time (22:00 UTC, 23:00 UTC, etc.) |
| **Room** | Group of 2-4 players (WMGT1, WMGT2, etc.) |
| **Easy/Hard** | The two courses played each week |
| **Scorecard** | In-game results screenshot |
| **Verification** | Confirming entered scores match screenshots |
| **Infraction** | Rule violation that may result in penalty |

## Penalty Rules

- **Infraction + Scores** → -1 season point
- **Infraction only (no scores)** → Week discarded
- **Multiple infractions** → Possible suspension
