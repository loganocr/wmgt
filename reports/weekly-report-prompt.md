# Weekly Tournament Report - Manual Prompt

## Quick Copy-Paste Prompt

```
Connect to wmgt database and analyze Week [REPLACE_WITH_WEEK] tournament results.

Generate a comprehensive tournament report including:

1. **Top 3 Podium** - Winners with stats and brief performance notes
2. **Week Statistics** - Total players, countries, score averages, rank distribution, course difficulty analysis  
3. **🌟 Player of the Week** - Someone OUTSIDE top 3 who significantly overperformed their rank (focus on 15+ position improvements vs rank average)
4. **Hidden Gems & Insights** - Rank upsets, international standouts, course mastery, balanced performances
5. **Country Performance** - Best representatives and interesting participation patterns
6. **Notable Achievements** - Special accomplishments, biggest surprises, fun statistics

Analysis Guidelines:
- Focus on stories behind numbers, not just statistics
- Highlight underdog performances and unexpected results  
- Use emojis to make it engaging
- Compare performance vs rank expectations
- Look for interesting patterns in country/rank performance
- Celebrate achievements at all levels

Make it engaging, positive, and highlight the human stories in the data!
```

## Usage Instructions

1. **Copy the prompt above**
2. **Replace [REPLACE_WITH_WEEK]** with actual week (e.g., S18W03)
3. **Paste into Kiro chat**
4. **Get comprehensive report in 2-3 minutes**
